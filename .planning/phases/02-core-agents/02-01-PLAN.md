    import pytest
    from unittest.mock import patch, AsyncMock
    from backend.models.agent_types import AgentResult


    def _mock_result(agent_name: str) -> AgentResult:
        return AgentResult(
            agent=agent_name,
            score=7.5,
            confidence=0.85,
            evidence=["Test evidence item"],
            flags=[]
        )

        return AgentResult(
            agent=agent_name,
            score=0,
            confidence=0,
            evidence=[],
            flags=["AGENT_UNAVAILABLE"]
        )

        assert result.agent == expected_agent
        assert 0.0 <= result.score <= 10.0
        assert 0.0 <= result.confidence <= 1.0
        assert isinstance(result.evidence, list)

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Python code → Claude API | Product dict serialized as JSON is sent to the LLM; untrusted product field values cross this boundary |
| LLM response → AgentResult | Claude's text response is parsed as JSON; malformed responses cross into application state |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | price/specs/review SYSTEM prompts | mitigate | System prompts constrain Claude to "Use ONLY the provided product data"; JSON serialization of product_context via json.dumps() prevents raw string injection |
| T-02-02 | Tampering | base.py json.loads(response.content[0].text) | mitigate | json.loads() inside try/except in base.py — malformed JSON returns AGENT_UNAVAILABLE, never propagates to caller |
| T-02-03 | Information Disclosure | ANTHROPIC_API_KEY | mitigate | Key loaded from .env via python-dotenv; .env in .gitignore; .env.example committed without real values |
</threat_model>
